## Rodando o projeto


### 1 - Clone o projeto do github e acesse o diretorio.

```bash
$ git clone https://github.com/matheusth/alurakut.git
$ cd alurakut
```
### 2 - Instale as dependencias.

```bash
$ npm install
# ou
$ yarn install
```

### 3 - Rode o projeto como desenvolvedor
```bash
$ yarn dev
# ou
$ npm run dev
```
